**Descripción del proyecto:**
ProfeYa es una aplicación web que conecta a estudiantes con profesores particulares de manera rápida y sencilla. El problema que intenta resolver es la dificultad que tienen los alumnos para encontrar docentes de confianza según la materia que necesitan, ya sea de forma virtual o presencial.

**Los usuarios principales son:**
Estudiantes que buscan apoyo académico en distintas materias.
Profesores particulares que desean ofrecer sus servicios y aumentar su visibilidad.

**Funcionalidades actuales:**
* Landing principal (index.html) – Página de bienvenida con carrusel de imágenes automático.
* Inicio de sesión de profesores (login-profesores.html) – Formulario de acceso para docentes registrados.
* Registro de profesores (registro-profesores.html) – Formulario para que los profesores creen su cuenta.
* Buscar profesores (buscar-profesores.html) – Vista con tarjetas de docentes disponibles, filtrable por materia y modalidad.

**Tecnologías utilizadas:**
HTML5
CSS3
JavaScript (vanilla)
Git & GitHub
